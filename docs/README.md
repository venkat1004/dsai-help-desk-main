## Document Organization Principles

- **Minimal Folder Depth:** Max 2 levels (docs/ → category/ → files)
- **Balanced Distribution:** 4-8 files per folder (except reference folders)
- **Clear Ownership:** Each folder has a README with owner and maintenance schedule
- **Cross-References:** Related documents linked in README files
- **Functional Grouping:** Files organized by purpose, not chronology
