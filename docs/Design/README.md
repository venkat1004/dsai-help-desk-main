## How to Use

**For UI Development:**
1. Review `Color-Palette.md` for official colors and usage guidelines
2. Reference `style-research.md` for design patterns and best practices
3. Apply to React MUI components in `src/components/`

**For Design Consistency:**
1. Use Color-Palette.md as single source of truth for colors
2. Ensure all new components follow style guidelines from `style-research.md`
3. Maintain consistency across all demo screens and production code

**For Design Reviews:**
1. Reference both documents during design review process
2. Ensure new designs align with established palette and guidelines
3. Document any design decisions or deviations
