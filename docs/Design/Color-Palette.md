# PCTE AI Help Desk - Color Palette

## Primary Colors

### Army Gold (Primary Brand Color)
- **Main**: `#D4AF37` - Primary gold, used for headers, buttons, highlights
- **Light**: `#E8C547` - Hover states, lighter accents
- **Dark**: `#B8941F` - Darker variant (if needed)

### Army Green (Secondary/Success)
- **Main**: `#4A7C59` - Success states, positive indicators, secondary actions
- **Light**: `#6BA876` - Light green accents
- **Dark**: `#2F5A3F` - Darker variant (if needed)

## Accent Colors

### Highlight Orange (Warning)
- **Main**: `#FF9500` - Warnings, alerts, attention-grabbing elements

### Red (Error/Critical)
- **Main**: `#D32F2F` - Errors, critical alerts (used sparingly)

### Blue (Info/External Systems)
- **Info Blue**: `#2196F3` - Information, external system integrations
- **Atlassian Blue**: `#0052CC` - Used for Jira/Confluence integrations

## Background Colors

### Dark Theme Backgrounds
- **Default Background**: `#1a1a1a` - Main page background
- **Paper/Card Background**: `#242424` - Cards, panels, elevated surfaces
- **AppBar/Header**: `#121212` - Top navigation bar
- **Border/Divider**: `#333333` - Borders, dividers, subtle separations

## Text Colors

### Text Hierarchy
- **Primary Text**: `#E0E0E0` - Main text content (off-white, 87% opacity equivalent)
- **Secondary Text**: `#999999` - Secondary text, labels, metadata (60% opacity equivalent)
- **Disabled Text**: `#666666` - Disabled/inactive text (38% opacity equivalent)

## Additional Colors Used

### Status Colors
- **Active/Online**: `#4A7C59` (Green)
- **Pending/Warning**: `#FF9500` (Orange)
- **Critical/Error**: `#D32F2F` (Red)
- **Neutral**: `#999999` (Gray)

### Chart Colors (Sentiment Analysis)
- **Neutral**: `#FF9500` (Orange)
- **Frustrated**: `#D32F2F` (Red)
- **Satisfied**: `#4A7C59` (Green)

## Color Usage Guidelines

### Primary Actions
- Use **Army Gold** (`#D4AF37`) for primary buttons, CTAs, and key highlights
- Hover state: `#E8C547`

### Success/Positive States
- Use **Army Green** (`#4A7C59`) for success messages, positive metrics, completion states

### Warnings/Alerts
- Use **Orange** (`#FF9500`) for warnings, important notices, medium-priority alerts

### Errors/Critical
- Use **Red** (`#D32F2F`) sparingly for critical errors and high-priority alerts

### Information
- Use **Blue** (`#2196F3`) for informational messages and external system references

## Design Philosophy

The color palette is designed for:
- **Dark theme** - Optimized for low-light environments and extended use
- **Military/Army aesthetic** - Gold and green reflect military heritage
- **High contrast** - Ensures accessibility and readability
- **Professional appearance** - Suitable for enterprise/government environments
